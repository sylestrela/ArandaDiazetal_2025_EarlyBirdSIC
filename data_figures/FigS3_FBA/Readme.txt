This repo contains data code to perform FBA analysis for Aranda Diaz et al (2024) Assembly of stool-derived bacterial communities follows “early-bird” resource utilization dynamics.  

To perform FBA analysis run: FBA.ipynb
To process raw OD Data run: Process_Raw_OD.R
To plot the two against each other and generate final figure run: Plot_OD_VS_FBA.R
